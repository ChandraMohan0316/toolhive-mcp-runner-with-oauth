toolhive-mcp GitHub

Helm chart for deploying any remote MCP server behind ToolHive's embedded OAuth 2.0 / Azure AD authentication proxy on Kubernetes (EKS + AWS ALB).

Install

helm repo add toolhive-mcp https://chandramohan0316.github.io/toolhive-mcp-runner-with-oauth
helm repo update
helm install <release-name> toolhive-mcp/toolhive-mcp \
  -n toolhive \
  -f my-values.yaml \
  --set server.bearerToken="<token>" \
  --set oauth.azure.clientSecret="<secret>"
